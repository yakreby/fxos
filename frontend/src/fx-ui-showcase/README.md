# fx-ui-showcase

`fx-ui` kütüphanesindeki tüm bileşenleri **canlı örnek + kullanım** ile tanıtan iç dokümantasyon arayüzü ("fx-debug").

Header'daki **fx-debug** butonundan açılır. İçerik:
- Her bileşenin canlı örneği ve varyantları (FxButton, FxBadge, FxAvatar, FxCard, FxStatCard, FxTable, FxToast)
- Tüm `FxIcon` ikon galerisi (`FX_ICON_NAMES`)
- `fx-` utility class referans tablosu

Amaç: Projede çalışan diğer geliştiricilerin bileşenleri hızlıca keşfedip doğru kullanması — Storybook benzeri ama kendi içimizde, sıfır dış bağımlılıkla.

> Yeni bir `fx-ui` bileşeni eklenince buraya da bir örnek bölümü eklenmeli.
